## Запуск

Из директории `3_task`:

```bash
python3 boolean_search.py
```

## Формат инвертированного индекса

Файл `inverted_index.txt`:

```text
<term> <doc_id1> <doc_id2> ... <doc_idN>
```

Каждая строка: один терм и отсортированный список документов, где он встречается.

## Файлы

- `boolean_search.py` — код построения индекса и булева поиска.
- `inverted_index.txt` — сгенерированный инвертированный индекс.
