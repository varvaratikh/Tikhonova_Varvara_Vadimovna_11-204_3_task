	<h1 id="firstHeading" class="firstHeading mw-first-heading"><span class="mw-page-title-main">Экономика</span></h1>
	<div id="bodyContent" class="vector-body">
		<div id="siteSub" class="noprint">Материал из Википедии — свободной энциклопедии</div>

		<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="ru" dir="ltr"><div role="note" class="hatnote navigation-not-searchable dabhide">Эта статья о хозяйственной деятельности; о науке см. <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0_(%D0%BD%D0%B0%D1%83%D0%BA%D0%B0)" title="Экономика (наука)">Экономика (наука)</a>.</div>
<div role="note" class="hatnote navigation-not-searchable noprint dabhide">У этого термина существуют и другие значения, см. <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" class="mw-disambig" title="Экономика (значения)">Экономика (значения)</a>.</div>
<p><b>Эконо́мика</b> (от <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;"><a href="https://ru.wiktionary.org/wiki/%CE%BF%E1%BC%B6%CE%BA%CE%BF%CF%82" class="extiw" title="wikt:οἶκος">οἶκος</a></span></span> «дом, хозяйство» + <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;"><a href="https://ru.wiktionary.org/wiki/%CE%BD%CF%8C%CE%BC%CE%BF%CF%82" class="extiw" title="wikt:νόμος">νόμος</a></span></span> «правило, закон»; <a href="/wiki/%D0%94%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%B2%D0%BE%D0%B4" title="Дословный перевод">дословно</a><span class="nowrap">&#8201;—</span> «правила ведения <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B5%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%BE" class="mw-redirect" title="Домашнее хозяйство">домашнего хозяйства</a>»)&#160;— <a href="/wiki/%D0%A5%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B4%D0%B5%D1%8F%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Хозяйственная деятельность">хозяйственная деятельность</a>, а также совокупность <a href="/wiki/%D0%9E%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BE%D1%82%D0%BD%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D1%8F" title="Общественные отношения">общественных отношений</a>, которые складываются в системе <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B8%D0%B7%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Производство">производства</a>, <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%80%D0%B5%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_(%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0)" title="Распределение (экономика)">распределения</a>, <a href="/wiki/%D0%9E%D0%B1%D0%BC%D0%B5%D0%BD_(%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0)" title="Обмен (экономика)">обмена</a> и <a href="/wiki/%D0%9F%D0%BE%D1%82%D1%80%D0%B5%D0%B1%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Потребление">потребления</a> товаров и услуг. В результате этой деятельности непрерывно воспроизводятся <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE_(%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0)" title="Благо (экономика)">блага</a>, обеспечивающие <a href="/wiki/%D0%96%D0%B8%D0%B7%D0%BD%D0%B5%D0%B4%D0%B5%D1%8F%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C#Жизнедеятельность_человека" title="Жизнедеятельность">жизнедеятельность людей</a><sup id="cite&#95;ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>Впервые слово «экономика» встречается в IV&#160;в. до&#160;нашей эры. <a href="/wiki/%D0%9A%D1%81%D0%B5%D0%BD%D0%BE%D1%84%D0%BE%D0%BD%D1%82" title="Ксенофонт">Ксенофонт</a> называет её «<a href="/wiki/%D0%95%D1%81%D1%82%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BD%D0%B0%D1%83%D0%BA%D0%B8" title="Естественные науки">естественной наукой</a>». <a href="/wiki/%D0%90%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%82%D0%B5%D0%BB%D1%8C" title="Аристотель">Аристотель</a> считал наиболее важным критерием правильного ведения хозяйства (экономики) нацеленность на создание благ для потребления и развития хозяйства. Он противопоставлял экономике <a href="/wiki/%D0%A5%D1%80%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Хрематистика">хрематистику</a>&#160;— деятельность, направленную исключительно на извлечение денежной <i><a href="/wiki/%D0%9F%D1%80%D0%B8%D0%B1%D1%8B%D0%BB%D1%8C" title="Прибыль">выгоды</a></i><sup id="cite&#95;ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>Экономические процессы представляют собой сложную и всеохватывающую систему, они протекают в противоречивых условиях ограниченных <a href="/wiki/%D0%A0%D0%B5%D1%81%D1%83%D1%80%D1%81%D1%8B" class="mw-redirect" title="Ресурсы">ресурсов</a> и безграничных <a href="/wiki/%D0%9F%D0%BE%D1%82%D1%80%D0%B5%D0%B1%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Потребность">потребностей</a> человека и общества.
</p>

<li><a href="/wiki/%D0%A7%D0%B5%D1%82%D0%B2%D0%B5%D1%80%D1%82%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D1%81%D0%B5%D0%BA%D1%82%D0%BE%D1%80_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8" title="Четвертичный сектор экономики">Четвертичный сектор экономики</a>&#160;— <a href="/wiki/%D0%98%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D1%82%D0%B5%D1%85%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D0%B8" title="Информационные технологии">информационные технологии</a>, <a href="/wiki/%D0%9E%D0%B1%D1%80%D0%B0%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Образование">образование</a>, <a href="/wiki/%D0%9D%D0%98%D0%9E%D0%9A%D0%A0" class="mw-redirect" title="НИОКР">научные исследования</a>, <a href="/wiki/%D0%93%D0%BB%D0%BE%D0%B1%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D1%80%D0%BA%D0%B5%D1%82%D0%B8%D0%BD%D0%B3" class="mw-redirect" title="Глобальный маркетинг">глобальный маркетинг</a>, <a href="/wiki/%D0%91%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B5_%D1%83%D1%81%D0%BB%D1%83%D0%B3%D0%B8" class="mw-redirect" title="Банковские услуги">банковские</a> и <a href="/wiki/%D0%A4%D0%B8%D0%BD%D0%B0%D0%BD%D1%81%D0%BE%D0%B2%D1%8B%D0%B5_%D1%83%D1%81%D0%BB%D1%83%D0%B3%D0%B8" title="Финансовые услуги">финансовые услуги</a> и другие услуги, связанные не с производством как таковым, а с его планированием и организацией (<a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0_%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B9" title="Экономика знаний">экономика знаний</a>).</li></ul>
<p>В зависимости от <a href="/wiki/%D0%A1%D0%BE%D0%B1%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Собственность">форм собственности</a> выделяют <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%81%D0%B5%D0%BA%D1%82%D0%BE%D1%80_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8" title="Государственный сектор экономики">государственный</a> и <a href="/wiki/%D0%A7%D0%B0%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%81%D0%B5%D0%BA%D1%82%D0%BE%D1%80_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8" title="Частный сектор экономики">частный</a> сектор.
</p><p>В зависимости от конкретных видов экономической деятельности различают <a href="/wiki/%D0%A0%D0%B5%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%81%D0%B5%D0%BA%D1%82%D0%BE%D1%80_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8" title="Реальный сектор экономики">реальный сектор экономики</a>, непроизводственный, финансовый.
</p>

<div class="mw-heading mw-heading2"><h2 id="Экономический_рост"><span id=".D0.AD.D0.BA.D0.BE.D0.BD.D0.BE.D0.BC.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B9_.D1.80.D0.BE.D1.81.D1.82"></span>Экономический рост</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Экономический рост»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Экономический рост»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<div role="note" class="hatnote navigation-not-searchable ts-main"><link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r142002967" />Основная статья: <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%BE%D1%81%D1%82" title="Экономический рост">Экономический рост</a></div>
<p>Экономический рост определяют как увеличение реального производства продукции в национальной экономике за определённый период времени (за месяц, квартал, год). В отличие от экономического развития, экономический рост&#160;— количественный показатель. Под реальным производством обычно понимают реальный (то есть очищенный от факторов инфляции) <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BD%D1%83%D1%82%D1%80%D0%B5%D0%BD%D0%BD%D0%B8%D0%B9_%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82" title="Валовой внутренний продукт">валовой внутренний продукт</a> (ВВП), реже&#160;— реальные <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BD%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82" title="Валовой национальный продукт">валовой национальный продукт</a> (ВНП), <a href="/wiki/%D0%A7%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D0%BD%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82" title="Чистый национальный продукт">чистый национальный продукт</a> (ЧНП), или <a href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B4%D0%BE%D1%85%D0%BE%D0%B4" title="Национальный доход">национальный доход</a> (НД). Экономический рост тесно связан с ростом общего благосостояния: ростом продолжительности жизни, качества медицинского обслуживания, уровня образования, сокращением продолжительности рабочего дня и&#160;т.&#160;д.
</p><p>Различают <a href="/wiki/%D0%AD%D0%BA%D1%81%D1%82%D0%B5%D0%BD%D1%81%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B2%D0%B5%D0%BB%D0%B8%D1%87%D0%B8%D0%BD%D0%B0" class="mw-redirect" title="Экстенсивная величина">экстенсивные</a> и <a href="/wiki/%D0%98%D0%BD%D1%82%D0%B5%D0%BD%D1%81%D0%B8%D0%B2%D0%BD%D1%8B%D0%B5_%D1%84%D0%B0%D0%BA%D1%82%D0%BE%D1%80%D1%8B_%D1%80%D0%B0%D0%B7%D0%B2%D0%B8%D1%82%D0%B8%D1%8F" class="mw-redirect" title="Интенсивные факторы развития">интенсивные</a> факторы роста экономики.
</p>

<p>В <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B1%D1%8B%D1%82%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Первобытное общество">первобытном обществе</a> уровень экономического развития был низким, обеспечивающим потребление на грани физического выживания. Сначала первобытные люди добывали средства к существованию <a href="/wiki/%D0%9E%D1%85%D0%BE%D1%82%D0%BD%D0%B8%D0%BA%D0%B8_%D0%B8_%D1%81%D0%BE%D0%B1%D0%B8%D1%80%D0%B0%D1%82%D0%B5%D0%BB%D0%B8" class="mw-redirect" title="Охотники и собиратели">охотой и собирательством</a>, но в результате <a href="/wiki/%D0%9D%D0%B5%D0%BE%D0%BB%D0%B8%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Неолитическая революция">неолитической революции</a> возникли <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D0%B5%D0%B4%D0%B5%D0%BB%D0%B8%D0%B5" title="Земледелие">земледелие</a> и <a href="/wiki/%D0%96%D0%B8%D0%B2%D0%BE%D1%82%D0%BD%D0%BE%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Животноводство">животноводство</a>. Развитие общества привело к <a href="/wiki/%D0%A0%D0%B0%D0%B7%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D1%82%D1%80%D1%83%D0%B4%D0%B0" title="Разделение труда">разделению труда</a>&#160;— выделились земледельческие и <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%82%D1%83%D1%88%D0%B5%D1%81%D0%BA%D0%B8%D0%B5_%D0%BF%D0%BB%D0%B5%D0%BC%D0%B5%D0%BD%D0%B0" title="Пастушеские племена">пастушеские племена</a>, выделились <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B5%D1%81%D0%BB%D0%BE" title="Ремесло">ремесленники</a>, первыми из которых были <a href="/wiki/%D0%9A%D1%83%D0%B7%D0%BD%D0%B5%D1%86" title="Кузнец">кузнецы</a>. Появились <a href="/wiki/%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B5%D1%80%D0%B0%D0%B2%D0%B5%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Социальное неравенство">социальное неравенство</a>, <a href="/wiki/%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BB%D0%B0%D1%81%D1%81" title="Социальный класс">социальные классы</a> и <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a>. Возникло <a href="/wiki/%D0%A0%D0%B0%D0%B1%D1%81%D1%82%D0%B2%D0%BE" title="Рабство">рабовладение</a>.
</p><p>Постепенно развивался товарообмен, который сначала осуществлялся в форме натурального обмена (<a href="/wiki/%D0%91%D0%B0%D1%80%D1%82%D0%B5%D1%80" title="Бартер">бартера</a>), но с появлением <a href="/wiki/%D0%94%D0%B5%D0%BD%D1%8C%D0%B3%D0%B8" title="Деньги">денег</a> превратился в <a href="/wiki/%D0%A2%D0%BE%D1%80%D0%B3%D0%BE%D0%B2%D0%BB%D1%8F" title="Торговля">торговлю</a>. Тем не менее, в обществах <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80" title="Древний мир">Древнего мира</a> и <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D0%B8%D0%B5_%D0%B2%D0%B5%D0%BA%D0%B0" title="Средние века">Средневековья</a> преобладающим было <a href="/wiki/%D0%9D%D0%B0%D1%82%D1%83%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Натуральное хозяйство">натуральное хозяйство</a>. Во многих государствах древности существовала так называемая <a href="/wiki/%D0%94%D0%B2%D0%BE%D1%80%D1%86%D0%BE%D0%B2%D0%B0%D1%8F_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0" title="Дворцовая экономика">дворцовая экономика</a>, основанная на сочетании планового хозяйства (позволяющего осуществлять крупные общественные работы, такие как <a href="/wiki/%D0%9E%D1%80%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D0%B5" title="Орошение">орошение</a>, сооружение дворцов и пирамид) и натурального хозяйства.
</p><p>С началом <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4%D1%8B" title="Крестовые походы">крестовых походов</a> европейцы заново открыли для себя <a href="/wiki/%D0%9F%D1%80%D1%8F%D0%BD%D0%BE%D1%81%D1%82%D0%B8" title="Пряности">пряности</a>, <a href="/wiki/%D0%A8%D1%91%D0%BB%D0%BA" title="Шёлк">шёлк</a> и другие товары, ставшие редкими в Европе в Средние века. Это открытие привело к <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BC%D0%B5%D1%80%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Коммерческая революция">расширению торговли и обострению торговой конкуренции со странами Востока</a>.
</p><p>С конца XV&#160;века началась <a href="/wiki/%D0%AD%D0%BF%D0%BE%D1%85%D0%B0_%D0%B2%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D1%85_%D0%B3%D0%B5%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D1%85_%D0%BE%D1%82%D0%BA%D1%80%D1%8B%D1%82%D0%B8%D0%B9" class="mw-redirect" title="Эпоха великих географических открытий">Эпоха великих географических открытий</a>, которая привела к тому, что сложилась <a href="/wiki/%D0%9C%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0" title="Мировая экономика">мировая экономика</a> и началась <a href="/wiki/%D0%AD%D0%BF%D0%BE%D1%85%D0%B0_%D0%BF%D0%B5%D1%80%D0%B2%D0%BE%D0%BD%D0%B0%D1%87%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BD%D0%B0%D0%BA%D0%BE%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Эпоха первоначального накопления">эпоха первоначального накопления</a>.
</p><p>С последней четверти XVIII&#160;века началась <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BC%D1%8B%D1%88%D0%BB%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Промышленная революция">промышленная революция</a>, которая привела к тому, что в наиболее развитых странах большинство населения к концу XIX&#160;века было занято уже не в сельском хозяйстве, а в <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BC%D1%8B%D1%88%D0%BB%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Промышленность">промышленности</a>. Преобладающей экономической системой стал <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Капитализм">капитализм</a>, происходил процесс превращения <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%B4%D0%B8%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Традиционное общество">традиционного общества</a> в <a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D1%80%D0%BD%D0%B8%D1%82%D0%B8" class="mw-redirect" title="Модернити">современное</a>, <a href="/wiki/%D0%90%D0%B3%D1%80%D0%B0%D1%80%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Аграрное общество">аграрного общества</a> в <a href="/wiki/%D0%98%D0%BD%D0%B4%D1%83%D1%81%D1%82%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Индустриальное общество">индустриальное общество</a>.
</p><p>В XX&#160;веке в ряде стран была создана <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B0%D0%BD%D0%B4%D0%BD%D0%B0%D1%8F_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0" class="mw-redirect" title="Командная экономика">административно-командная</a> <a href="/wiki/%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Социализм">социалистическая</a> экономика. В остальных странах происходило развитие капитализма. Во второй половине XX&#160;века началась <a href="/wiki/%D0%9D%D0%B0%D1%83%D1%87%D0%BD%D0%BE-%D1%82%D0%B5%D1%85%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Научно-техническая революция">научно-техническая революция</a>, в результате которой индустриальное общество в наиболее развитых странах стало превращаться в <a href="/wiki/%D0%9F%D0%BE%D1%81%D1%82%D0%B8%D0%BD%D0%B4%D1%83%D1%81%D1%82%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Постиндустриальное общество">постиндустриальное</a>. В ряде стран, в том числе в <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a><sup id="cite&#95;ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup>, переход к постиндустриальной экономике тормозится в силу ряда объективных и субъективных причин.
</p>
